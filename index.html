iTELL Knowledge Base Mockups

Select a version to preview. Each version shares the same base styles but explores a different layout and information architecture for the user-facing help center.

Version 1 — Help Center with Sidebar Nav

Category-based knowledge base with persistent sidebar navigation, hero search, article cards, FAQ accordion, step-by-step guides, callout boxes, and "was this helpful" feedback widgets. Covers all core iTELL features: reading, summaries, questions, cloze, iTELL AI, dashboard, notes, and data privacy.
9 pages: Home · Getting Started · Reading & Progress · Summaries · Questions & Cloze · iTELL AI · Dashboard · Notes & Highlights · FAQ · Data & Privacy

Version 2 — Multi-Audience + Textbook-Aware

Builds on v1 with three key additions: (1) Volume switcher in the topbar lets users filter content by textbook (Demo, Chevron, Mars) — articles and sections that don't apply are hidden. (2) Audience tabs in the sidebar switch between Student, Instructor, and Content Author views. (3) Conditional content blocks within articles show/hide sections based on textbook selection (e.g., CRI+Cloze for Demo/Chevron, Quizzes for Mars).
12 pages: Home · Summaries · Quizzes (Mars) · Instructor Overview · Class Setup · Student Progress · Instructor FAQ · Authoring Overview · Content Structure · Payload Blocks · Author FAQ
