iTELL Teacher Dashboard Mockups

Select a version to preview. Each version shares the same base styles but explores a different layout and information architecture.

Version 1 — Classic SaaS

Volume-centric layout with sidebar navigation. Workforce training context (employees, modules). Dynamic tabs for Summaries, Quick Questions, and Cloze that appear based on volume configuration. End-of-module activities vary per module (summary or cloze). Leaderboard with streak-based rankings, quiz results, and full employee drill-down.
5 pages: Overview · Summaries · Quick Questions · Cloze · Employee Detail

Version 2 — Manager Executive Summary

KPI-forward layout with sparklines, trend indicators, and period selectors. Designed for managers who need a 30-second status check. Large headline numbers with week-over-week trends, progress rings, top/bottom performer cards, and activity pass rate table with directional arrows.
5 pages: Overview · Summaries · Quick Questions · Cloze · Employee Detail

Version 3 — Action-Oriented / Intervention

Surfaces problems first with alert banners, risk pills, and failure-focused tables. At-risk employees and struggling modules are highlighted with recommended actions. Designed for training administrators who need to know what requires their attention right now.
5 pages: Overview · Summaries · Quick Questions · Cloze · Employee Detail
